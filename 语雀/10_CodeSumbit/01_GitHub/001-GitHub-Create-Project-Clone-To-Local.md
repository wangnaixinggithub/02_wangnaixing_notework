# GitHub-Create-Repository-Clone-To-Local-Do-Submit-Local-Project

# 1、Create Repository

```she
# 1、在官网中注册 GitHub 然后以此个人网店执行登录。
https://github.com/wangnaixinggithub

#2、进入创建页面，需要你本人确定仓库的名称

#3、可选性的确定仓库的说明信息

#4、点选此仓库是公有仓库还是私有仓库

#5、为这个仓库添加一个REAMDE文件

#6、选择此仓库的许可说明 

#7、点击确认，即可完成仓库的创建
```

点击`Code` 按钮，在弹出的下拉框中，复制此克隆地址。因为本人安装了浏览器插件，下方显示的是镜像下载。

# 4、Open Git Bash And Do Clone

找到一个合适的位置，右键开启`git bush` 模式，进入`git` 控制台。

```
# 1、执行git clone 命令 进行远程仓库加载到本地
git clone https://gitclone.com/github.com/wangnaixinggithub/01_wangnaixing_security.git

# 2、得到.git版本控制文件 README.md 仓库说明文件  LICENSE 许可说明文件 .gitignore 提交忽略文件

#3、将上述全部文件复制到你需要提交的本地工程文件中
```
