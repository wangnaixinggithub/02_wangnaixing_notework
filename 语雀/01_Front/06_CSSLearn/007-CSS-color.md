# CSS-color

- 我们可以通过color属性，设置盒子内文字的颜色，比如当下我设置盒子颜色为红色。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<style>
.box1{
    width: 150px;
    height: 150px;
    text-align: center;
    border: 1px solid red;
    /*设置文字的颜色*/
    color: red;

}
</style>

<body>
<div class="box1">
    淘宝二维码
</div>
</body>
</html>
```
