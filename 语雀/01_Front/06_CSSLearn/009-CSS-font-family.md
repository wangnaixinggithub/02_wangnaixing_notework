# CSS-font-family

- 通过font-family属性，可以设置盒子内字体的样式。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<style>
.box1{
    width: 150px;
    height: 150px;
    text-align: center;
    border: 1px solid red;
    color: red;
    /*设置盒子字体样式*/
    font-family: '宋体',system-ui;

}
</style>

<body>
<div class="box1">
    淘宝二维码
</div>
</body>
</html>
```
