# CSS-font-size

- 通过`font-size`属性可以直接设置修改盒子里面的文本大小

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<style>
.box1{
    width: 150px;
    height: 150px;
    text-align: center;
    border: 1px solid red;
    color: red;
    /*设置盒子字体大小*/
    font-size: 28px;

}
</style>

<body>
<div class="box1">
    淘宝二维码
</div>
</body>
</html>
```
