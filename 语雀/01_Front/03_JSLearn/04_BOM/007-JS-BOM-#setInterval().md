# JS-BOM-#setInterval()

- 设置一个重复执行的定时器。

```html
<!DOCTYPE html>
<html lang="en">

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
    <style></style>
</head>
<body>

<script>
    //反复调用.... 
    let timer = setInterval(()=>{console.log('地球爆炸了,哈哈哈哈....')},1000)
</script>
</body>
</html>
```
