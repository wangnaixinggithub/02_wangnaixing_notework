# JS-Array-De-Construct

- JS 数组的解构，更为简洁的获取到数组元素

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Array-Method-De-Construct</title>
</head>
<body>
<script>
let arr = ['黄瓜','香蕉','西红柿']

const [a,b,c] = arr

console.log(a) //'黄瓜'
console.log(b) // '香蕉'
console.log(c) // '西红柿'

</script>
</body>
</html>
```
