# JS-NumberVariable-If-TrueOrFalse

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>JS-NumberVariable-If-TrueOrFalse</title>
</head>
<body>
<script>
let num1 = 1
let num2 = -1
let num3 = 0


if (num1){
    console.log('真')
}else {
    console.log('假')  //1.Result is 真
}


if (num2){
    console.log('真')
}else {
    console.log('假')  //2.Result is 真
}


if (num3){
    console.log('真')
}else {
    console.log('假')  //3.Result is 假
}

//总结：number变量，只有不为0就是真。
</script>
</body>
</html>
```
