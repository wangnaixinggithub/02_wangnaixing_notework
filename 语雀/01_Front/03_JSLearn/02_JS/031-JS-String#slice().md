# JS-String-#slice()

> 使用slice()方法可以便捷进行字符串从后往前提取

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<script>
    let arr = 'abcdef'
    console.log(arr.slice(-1)); //f
    console.log(arr.slice(-2)); //ef
    console.log(arr.slice(-3)); //def
</script>
</body>
</html>
```
