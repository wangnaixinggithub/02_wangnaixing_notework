# Js-Create-Array

# 1、[ ]

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Js-Create-Array</title>
</head>
<body>
<script>
   let myarr = ['aaa','bbb','ccc']
   
   console.log(myarr.length)
</script>
</body>
</html>
```

# 2、new Array

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Js-Create-Array</title>
</head>
<body>
<script>
   let myarr =  new Array('aaa','bbb','ccc')

   console.log(myarr.length)
</script>
</body>
</html>
```

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Js-Create-Array</title>
</head>
<body>
<script>
   let myarr =  new Array()
   myarr[0] = 'aaa'
   myarr[1] = 'bbb'
   myarr[2] = 'ccc'
   console.log(myarr.length)
</script>
</body>
</html>
```
