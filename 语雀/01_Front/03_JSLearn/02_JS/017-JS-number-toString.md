# JS-number-toString

- 将number数值类型，转换为字符串类型。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>JS-number-toString</title>
</head>
<body>
<script>
   let num1 = 4.154614754
   let str = num1.toString()

   console.log(Object.is('string',typeof str))
   console.log(Object.is('4.154614754',str))
</script>
</body>
</html>
```
