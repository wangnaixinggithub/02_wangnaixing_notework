# Python-range()

- 根据给定范围，创建出指定元素的列表。

```python
def main():
    arr = range(0, 5)

    print(arr[0])
    print(arr[1])
    print(arr[2])
    print(arr[3])
    print(arr[4])


if __name__ == '__main__':
    main()

```

```python
0
1
2
3
4
```

- 指定范围步长。

```python
def main():
    arr = range(0, 10, 2)

    print(arr[0])
    print(arr[1])
    print(arr[2])
    print(arr[3])
    print(arr[4])


if __name__ == '__main__':
    main()

```

```python
0
2
4
6
8
```
