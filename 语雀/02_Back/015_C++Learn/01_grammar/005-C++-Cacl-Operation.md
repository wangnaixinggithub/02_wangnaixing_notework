# C++-Cacl-Operation

```c++
#include <iostream>
using namespace std;
int main() {
    int number1 = 10;
    int number2 = 3;
    cout<<number1/number2<<endl; //3 两个整数相除，结果还是整数。

    float f1 = 10.0;
    float f2 = 3.0;

    cout<<f1/f2<<endl; //3.33333 两个小数相除 结果还是小数


    int d1 = 3.33;
    int d2 = 3.33;

    cout<<(d1==d2)<<endl; //1  C和C++ 语言的比较运算中， ==“真”用数字“1”来表示， “假”用数字“0”来表示.



}
```
