# Install-MySQL 8-Software-Way-2

> 上一种方式是解压安装，MySQL官方推出了Installer安装器，让安装图形化。更容易安装上手！

# 1、Download

进入官网，选择好MySQL的版本，以及使用的操作系统。下载2.3M的安装器`.msi`文件,并打开。

```
官网：https://downloads.mysql.com/archives/installer/
```
