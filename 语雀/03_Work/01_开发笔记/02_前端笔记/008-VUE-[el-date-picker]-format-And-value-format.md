# VUE-[el-date-picker]-format-And-value-format



使用日期组件el-date-picker，属性`format`，`value-format`数据模型日期对象Date变成日期字符串形式。 

```vue
 <el-date-picker
          id="SQGZKSSJ"
          v-model="primaryData.SQGZKSSJ"
          type="datetime"
          format="yyyy-MM-dd HH:mm:ss"
          value-format="yyyy-MM-dd HH:mm:ss"
          placeholder="选择日期时间">
 </el-date-picker>
```
