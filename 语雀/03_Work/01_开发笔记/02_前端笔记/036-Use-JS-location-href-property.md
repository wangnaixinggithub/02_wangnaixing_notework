# Use-JS-location-href-property

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>07_JS的内置对象location的href()方法</title>
</head>
<body>
<script>




     console.log(location.href)      // = 浏览器地址栏的网址URL

     //如果对其进行赋值修改则会跳转到此URL ,比如访问百度
     location.href = 'https://14.215.177.38:443/'



</script>
</body>
</html>
```
