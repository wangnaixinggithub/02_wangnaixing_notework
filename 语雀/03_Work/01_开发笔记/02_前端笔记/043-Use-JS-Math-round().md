# Use-JS-Math-round()

- 返回四舍五入后，最接近的整数。

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>15_Math的round函数</title>
</head>
<body>
<script>

    let num1 = 0.1
    let num2 = 1.1
    let num3 = 1.6
    let num4 = -1.1
    let num5 = -1.6
    let num6 = 1

    console.log(Math.round(num1)) //0
    console.log(Math.round(num2)) //1
    console.log(Math.round(num3)) //2
    console.log(Math.round(num4)) //-1
    console.log(Math.round(num5)) //-2
    console.log(Math.round(num6)) //1

</script>
</body>
</html>
```
